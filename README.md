# Internship-15-Movie-Explorer

Movie Explorer is a single-page application built with React that allows users to browse, search, sort, and favorite movies.

## Features

- Browse through 20 amazing movies
- Search movies by title using the built in search bar
- Sort movies alphabetically, by year of release or by IMDb rating using the dropdown menu
- Click on a movie to see additional information
- Favorite and unfavorite movies
- Favorites are saved in local storage so they persist through refreshes

## Usage

1. Clone or download the repository from GitHub
2. Inside a terminal (recommend using VS code and opening terminal using CTRL+J or Cmd+J on MacOS) run the following command to install all the dependencies
```
npm install
```
3. Start the server using 
```
npm run dev
```
4. Copy the url from the terminal into browser url search tab
